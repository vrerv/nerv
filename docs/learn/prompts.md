사용자는 어떤 단어가 자음과 모음이 랜덤하게 캔버스에 흩어져있을때, 제시된 원래 단어로 합쳐야 된다. 아래는 어떻게 자음과 모음을 사용자가 캔버스에서 드래그앤 드롭으로 조합할때, 단어로 인식하는 방법을 설명한다.

- 한글은 자음과 모음으로 구성된다

```javascript
  const KOR_INITIAL_LETTERS = ['ㄱ', 'ㄲ', 'ㄴ', 'ㄷ', 'ㄸ', 'ㄹ', 'ㅁ',
  'ㅂ', 'ㅃ', 'ㅅ', 'ㅆ', 'ㅇ', 'ㅈ', 'ㅉ',
  'ㅊ', 'ㅋ', 'ㅌ', 'ㅍ', 'ㅎ'];
  // 수직 모음
  const KOR_VERTICAL_VOWELS = ['ㅏ', 'ㅐ', 'ㅑ', 'ㅒ', 'ㅓ', 'ㅔ', 'ㅕ',
  'ㅖ', 'ㅣ'];
  // 수평 모음
  const KOR_HORIZONTAL_VOWELS = ['ㅗ', 'ㅜ', 'ㅛ', 'ㅠ', 'ㅡ'];
  // 대각 모음
  const KOR_BOTH_VOLES = ['ㅘ', 'ㅙ', 'ㅚ', 'ㅝ', 'ㅞ', 'ㅟ', 'ㅢ']
  const KOR_VOWELS_CHANGE = 28;
  const KOR_TERMINALS = ['', 'ㄱ', 'ㄲ', 'ㄳ', 'ㄴ', 'ㄵ', 'ㄶ',
  'ㄷ', 'ㄹ', 'ㄺ', 'ㄻ', 'ㄼ', 'ㄽ', 'ㄾ',
  'ㄿ', 'ㅀ', 'ㅁ', 'ㅂ', 'ㅄ', 'ㅅ', 'ㅆ',
  'ㅇ', 'ㅈ', 'ㅊ', 'ㅋ', 'ㅌ', 'ㅍ', 'ㅎ'];
```
- KOR_INITIAL_LETTERS 와 KOR_TERMINALS 는 자음중에 초성과 종성을 나타내고 같은 문자는 같은 문자가 초성과 종성에 같이 쓰일 수 있음을 의미한다.
- 각 문자는 4면중 하나와 연결될수 있고 두 문자를 가까이 가져가면 한 음절로 만들수 있다
- 자음끼리는 연결될 수 없다.
- 초성은 반드시 수직 모음의 왼쪽 또는 수평 모음의 위에 있어야 한다.
- 종성은 반드시 모음 아래에 있어야 한다.
- 자음은 초성 또는 종성에 위치할수 있으므로 위, 오른쪽, 아래 면에 모음과 결합할수 있다
- 모음은 수직 모음, 수평 모음, 대각 모음이 있을수 있다
- 모든 모음은 아래쪽으로 자음과 결합가능하다
- 수직 모음은 왼쪽에 수평 모음은 위에 대각 모음은 왼쪽 또는 위에 초성 자음과 연결될 수 있다
- 자음과 모음 문자가 흩어져 있을때, 사용자가 어떤 하나의 문자를 드래그 드롭하면, 드롭된 문자와 다른 문자가 연결될 수 있는지 검사한다.
- 두개의 자음이 연결되서 음절이 되면, 음절리스트에 음절이 캔버스에 위치한 순서대로 넣는다.
- 음절리스트가 단어와 일치하면 정답이된다.

위 조건을 만족하는 함수 다음 함수를 만들어라

* 사용자가 문자를 드롭했을때, 호출될 evalLetters(i, elements) 함수
* i 는 현재 드롭된 문자의 index 이다.
* element 는 드롭된 문자의 Shape 이다 Shape 는 { x, y, width, height, value } 를 가진다.
* return 으로 연결된 element index 를 음절단위의 리스트로 반환한다. 
  음절은 최대 3개의 문자를 가질 수 있다. 0번째는 초성, 1번째는 항상 모음, 2번째는 종성 자음이 온다. 예) [[0, 1], [2, 3, 4]]
