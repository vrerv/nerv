## 어디에 사용하나?

로컬 개발환경에서 HTTPS를 적용해서 테스트 하고 싶을 때 사용할 수 있는 방법 이다. 물론 대부분 HTTPS 환경은 외부에서 접속하는 환경이기 때문에 로컬 개발환경에서 HTTPS를 적용할 일이 없을 수도 있다.
또한 HTTPS를 적용할때 self-signed 인증서를 사용하지 않고 실제 운영 인증서를 사용하면 된다.

하지만 다음과 같은 상황이 있을 수 있다.

- 개발환경 설정이 안되어 로컬 구동시 운영환경으로 연결되는 경우에 서버 인증서를 로컬에 가져올 수 없는 경우
- 로컬에서 서버 https 를 접속하거나, 로컬 서버에 접속하는 것을 스위칭하고 싶을 경우
