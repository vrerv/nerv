OpenAI’s Tour 2023 London 에서 개발자와의 대화내용에서 밝혔다고 하는 2023~2024 로드맵

- 더 저렴하고 더 빠른 GPT-4 - 최우선 사항
- 더 긴 콘텍스트 윈도우 - 1백만 토큰 윈도우가 가까운 미래에 실현 될 것으로 전망
- 파인튜닝 API - 파인튜닝 API가 최신 모델까지 확장될 것으로 예상
- Stateful API - 현재 챗 API는 대화 내용을 전부 보내 맥락을 기억해야 하지만 미래에는 대화 이력을 기억하는 API가 나올 예정
- 2024년, 멀티 모달리티 - 이미지 등 멀티미디어 지원. GPT-4 출시 일부로 시연됐지만, 더 많은 GPU 를 확보하기 전까지는 모든 사용자에게 오픈 할 수 없음

그외

- 고객과 경쟁할 생각이 없다 - 챗GPT 외에는 다른 제품을 출시하지 않을 것
- 챗GPT 의 비전은 초똑똑한 비서가 되는 것 - 챗GPT는 플랫폼과 허브 역할만 할 테니 마음껏 갖다 써 새로운 AI 서비스를 만들라는 것
- 확장법칙(Scaling laws)를 옹호 - 내부 데이터를 갖고 있는데, 모델 크기는 몇 배로 증가하기 보다 매년 크기가 두배나 세배 커질것으로 예상
